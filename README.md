# Flask Artstore

## Setup (quick)
1. Install MySQL and create a user if needed.
2. Copy `.env.example` to `.env` and update DB credentials.
3. Create virtualenv and install requirements: `pip install -r requirements.txt`
4. Run SQL: `mysql -u root -p < create_database.sql`
5. Start app: `python app.py`
6. Open http://127.0.0.1:5000
